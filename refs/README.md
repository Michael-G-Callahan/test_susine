# Reference Notes

This folder keeps project notes organized by document role.

## Folders

- `background/`: conceptual and technical background for SuSiE, SuSiE-2, SuSiNE, and simulation design.
- `decisions/`: dated decision memos and execution handoffs.
- `study_plans/`: experiment and paper-study plans.
- `investigation/`: exploratory analyses, theory notes, and question-driven investigations.
- `methods_traces/`: detailed result-provenance traces for paper methods and reviewer-level details.
- `progress_notes/`: current status, task plans, and operational progress notes.
- `ideas/`: out-of-scope or future-work ideas.
- `old/`: archival notes retained for history, not current source of truth.
